CSS Practice: Colors, Object-fit, and Position

Color Formats

Examples of using different color formats in CSS:

  • RGB Red (rgb(255, 0, 0))
  • RGB Green (rgb(0, 255, 0))
  • RGB Blue (rgb(0, 0, 255))
  • RGBA with transparency (rgba(100, 150, 200, 0.5))
  • Custom Hexadecimal (#6495ED)
  • Custom HSL (hsl(210, 50%, 60%))
  • HSLA with transparency (hsla(210, 50%, 60%, 0.5))
