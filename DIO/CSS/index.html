CSS Color Formats: RGB, Hex, HSL

Examples of using different color formats in CSS:

  • RGB Red (rgb(255, 0, 0))
  • RGB Green (rgb(0, 255, 0))
  • RGB Blue (rgb(0, 0, 255))
  • Custom RGB (rgb(100, 150, 200))
  • Custom RGBA with transparency (rgba(100, 150, 200, 0.5))
  • Hexadecimal Blue (#0000FF)
  • Custom Hexadecimal (#6495ED)
  • HSL Yellow (hsl(60, 100%, 50%))
  • Custom HSL (hsl(210, 50%, 60%))
  • Custom HSLA with transparency (hsla(210, 50%, 60%, 0.5))
